### 自用博客系统
ysx-blog由thymeleaf + Mybatis-plus + SpringBoot等技术实现的Java系统，分为前台展示和后台管理界面。页面美观、功能齐全
部署简单及非常完善的代码以及相关说明均可在swagger中进行查看。引用了开源的数据库数据对数据进行填充。
